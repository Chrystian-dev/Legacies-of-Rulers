## 🚀 Tecnologias

Esse projeto foi desenvolvido com as seguintes tecnologias:

- HTML e CSS
- JavaScript
- Git e Github
- IA
- Canva

## 💻 Projeto

Interface de RPG de fantasia para web desenvolvida com tecnologias front-end (HTML5, CSS3 e JS). O projeto destaca a integração de assets visuais criados via IA e diagramados no Canva, resultando em um layout dinâmico que traz o universo fantástico para o ambiente dos navegadores de computador.
